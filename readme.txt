== Description ==
This plugin is only used with Styled Store a Free WordPress Theme to set up a demo. https://wordpress.org/themes/styled-store/. This plugin entertain feature from One Click Demo Import to import data, so first you must install One Click Demo Import plugin: https://wordpress.org/plugins/one-click-demo-import/


== Installation ==

1. Install and activate styledstore Demo Importer Plugin.
2. Install all plugin required by Demo Importer Plugin.
3. Be sure you have Styled Store themes Installed Download Link: https://wordpress.org/themes/styled-store/
4. Go to Appearace and find Import Demo Data submenu
5. Click on Demo data and your demo is ready

== Frequently Asked Questions ==
= Is this plugin used with other Styled Themes ? =
No, it is only used with Styled Store Free version
